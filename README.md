This repository contains my solutions and technical walkthroughs for the **OverTheWire: Bandit** wargame. Each level is documented in its own folder to show the specific commands and logic used to find the password.

## 📂 Repository Structure
* **Level-00 to Level-34**: Each directory contains a `README.md` explaining the "grind"—the specific man pages I read and the terminal tricks I used to bypass each hurdle.

## 💻 Tech Stack
* **OS**: Ubuntu 24.04
* **Machine**: Aorus Laptop (Dual-boot Windows 11)
* **Shell**: Bash
* **Key Tools**: `ssh`, `grep`, `find`, `xxd`, `base64`, `tr`

## 🛠 Solving Approach
I don't look for spoilers. I focus on:
* Digging through **man pages** (`man find`, `man xxd`, etc.).
* Dealing with shell friction (hidden files, special characters, nested compression).
* Scripting/Piping commands to automate repetitive searches.
